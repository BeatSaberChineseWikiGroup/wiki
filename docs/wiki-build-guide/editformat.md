# 编辑格式指南【编辑者】

这里记录了维基编辑所需的格式。

<提示>关于Docusaurus原本可用的格式内容，请参考[官方文档](https://docusaurus.io/docs)。</提示>

<提示>为了你能获得流畅的中文编写体验，这个维基的所有组件都是基于中文定制的！</提示>

此维基由docusaurus驱动，但为了降低维基编辑门槛，**所有的扩展组件**都使用中文名字进行二次开发。

点击页面上的编辑按钮即可编辑，在此之前，请确保你已经加入维基编辑组，请在Github上通过issue联系我们或加入QQ群！

## 提示框系列

标题可以省略，内容可以内联

### 语法说明

```html

<备注>示例内容</备注>
<备注 内容="示例内容"/>
<备注 标题="示例标题" 内容="示例内容"/>
<备注 title="示例标题" content="示例内容"/>
<备注 标题="示例标题">示例内容</备注>

<提示 标题="示例标题">示例内容</提示>
<信息 标题="示例标题">示例内容</信息>
<危险 标题="示例标题">示例内容</危险>
<警告 标题="示例标题">示例内容</警告>
<错误 标题="示例标题">示例内容</错误>
```

以上内容等价于markdown语法，包括`note` `tip` `info` `danger` `caution`

```
:::tip 示例标题
示例内容
:::
```

### 示例

```html
<提示>示例内容1</提示>
<提示 标题="示例标题" 内容="示例内容2"/>
<错误 标题="示例标题">示例内容3</错误>
```

<提示>示例内容1</提示>
<提示 标题="示例标题" 内容="示例内容2"/>
<错误 标题="示例标题">示例内容3</错误>
